# 🚀 Professional QR Studio

A modern, fully customizable QR Code generator built with Vanilla JavaScript and QuickChart API. This tool allows users to create high-quality QR codes with custom colors, logos, and border radius.

## ✨ Features

* **Real-time Customization:** Change colors, margins, and border radius with live preview.
* **Logo Support:** Upload your own logo (PNG/JPG) to place it in the center of the QR code.
* **Resolution Control:** Adjust the preview and download resolution from 100px up to 1000px.
* **Ultra-HD Download:** Download your customized QR code as a high-quality PNG with all styles (including border radius) preserved.
* **Responsive UI:** Dark-themed, modern interface built with Inter font and FontAwesome icons.

## 🛠️ Built With

* **Frontend:** HTML5, CSS3 (Flexbox/Grid, Backdrop-filter)
* **Scripting:** Vanilla JavaScript (ES6+)
* **API:** [QuickChart QR Code API](https://quickchart.io/qr-codes/)
* **Deployment:** GitHub Pages (via GitHub Actions)

## 🚀 How to Use

1.  **Enter Content:** Type your URL or text in the "Content" box.
2.  **Upload Logo:** (Optional) Select a logo file from your device.
3.  **Customize:** Use the sliders to adjust the Margin, Border Radius, and Resolution. Pick your favorite foreground and background colors.
4.  **Generate:** Click the **Generate** button to see your masterpiece.
5.  **Save:** Click **Save PNG** to download a high-resolution version of your QR code.
